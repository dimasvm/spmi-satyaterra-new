# Pengantar

Panduan ini menjelaskan cara memakai aplikasi SPMI untuk pekerjaan harian di panel admin.
Fokusnya adalah alur kerja, status yang perlu diperhatikan, dan menu yang dipakai sesuai role.

## Yang dibahas

- cara membaca dashboard;
- perbedaan `Monitoring Capaian` dan `Inbox Validasi Capaian`;
- langkah kerja untuk Admin LPM, Unit/PIC, dan Auditor;
- pertanyaan yang sering muncul saat memakai aplikasi.

## Alur baca yang disarankan

1. Baca [Dashboard dan navigasi](/dashboard-dan-navigasi).
2. Baca halaman sesuai role Anda:
   - [Admin LPM](/panduan/admin-lpm)
   - [Unit/PIC](/panduan/unit-pic)
   - [Auditor](/panduan/auditor)
3. Jika hanya ingin memahami menu capaian, baca:
   - [Monitoring Capaian](/pelaksanaan/monitoring-capaian)
   - [Inbox Validasi Capaian](/pelaksanaan/inbox-validasi-capaian)
4. Cek [FAQ](/faq) untuk jawaban singkat.

:::note
Dokumentasi ini ditulis untuk pengguna internal aplikasi dan mengikuti istilah yang dipakai di panel SPMI.
:::
