# Monitoring Capaian

`Monitoring Capaian` adalah halaman untuk melihat seluruh data capaian indikator yang masuk ke sistem.
Halaman ini dipakai untuk observasi, bukan untuk mengambil keputusan validasi.

## Isi yang ditampilkan

- indikator, unit, dan periode;
- target dan realisasi;
- status capaian;
- status submit;
- review terakhir;
- catatan review;
- tanggal dan pengirim data.

## Filter yang tersedia

- periode SPMI;
- status capaian;
- status submit.

## Kegunaan utama

1. Mengecek progres capaian lintas unit.
2. Melihat capaian mana yang sudah selesai, dikembalikan, atau belum ditindaklanjuti.
3. Membaca riwayat data sebelum masuk ke review.

## Yang bukan tugas halaman ini

- mengubah status capaian;
- memberi keputusan validasi;
- menulis catatan review final.

Untuk keputusan validasi, gunakan [Inbox Validasi Capaian](/pelaksanaan/inbox-validasi-capaian).
