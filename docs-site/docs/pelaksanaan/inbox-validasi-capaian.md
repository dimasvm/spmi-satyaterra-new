# Inbox Validasi Capaian

`Inbox Validasi Capaian` adalah antrian kerja Admin LPM untuk memeriksa capaian yang sudah dikirim unit.
Halaman ini dipakai saat LPM perlu memutuskan apakah capaian diterima, dikembalikan, atau ditolak.

## Status yang muncul

- `Menunggu Validasi`
- `Dikembalikan`
- `Tervalidasi`
- `Semua`

## Langkah kerja

1. Buka halaman inbox validasi.
2. Pilih periode dan tab status yang ingin dikerjakan.
3. Buka satu capaian.
4. Periksa realisasi, catatan, dan bukti.
5. Ambil keputusan:
   - **Validasi** jika data lengkap dan sesuai;
   - **Kembalikan** jika unit perlu memperbaiki data atau bukti;
   - **Tolak** jika data tidak dapat diterima.
6. Simpan keputusan agar status capaian dan status assignment ikut berubah.

## Bedanya dengan Monitoring Capaian

- `Monitoring Capaian` dipakai untuk melihat data.
- `Inbox Validasi Capaian` dipakai untuk memproses data.
- Monitoring bersifat observasi, inbox bersifat aksi.

Jika Anda hanya ingin memeriksa progres, buka [Monitoring Capaian](/pelaksanaan/monitoring-capaian). Jika ingin memberi keputusan, buka halaman ini.
