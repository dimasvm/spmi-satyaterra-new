# Dashboard dan Navigasi

Dashboard menampilkan ringkasan pekerjaan sesuai role. Gunakan dashboard untuk melihat item yang perlu dikerjakan lebih dulu.

## Grup menu utama

- **Penetapan**: standar, indikator, dokumen mutu, dan assignment.
- **Pelaksanaan**: capaian indikator, monitoring capaian, dan inbox validasi.
- **Evaluasi AMI**: jadwal audit, checklist, hasil audit, dan ruang kerja auditor.
- **Pengendalian**: temuan, monitoring temuan, tindak lanjut, dan verifikasi tindak lanjut.
- **Peningkatan**: rapat tinjauan manajemen dan usulan perbaikan standar.
- **Laporan**: laporan rekap dan ekspor.
- **Master Data**: unit, user, dan periode.

## Cara memakai navigasi

1. Mulai dari dashboard untuk melihat pekerjaan paling mendesak.
2. Gunakan menu grup sesuai proses yang sedang dikerjakan.
3. Buka halaman detail hanya untuk data yang memang sedang Anda proses.
4. Untuk panduan cepat, gunakan tombol panduan di topbar aplikasi.

## Kapan kembali ke dashboard

- setelah menyimpan perubahan;
- setelah mengirim capaian untuk validasi;
- setelah memfinalisasi audit atau tindak lanjut;
- saat ingin mengecek ringkasan status terbaru.
